= Remover uma Série E do NetApp Console
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Remova um sistema E-Series se não quiser mais gerenciá-lo no NetApp Console.

.Função necessária do NetApp Console
Administrador da organização, administrador da pasta ou do projeto ou administrador do armazenamento. link:https://docs.netapp.com/us-en/bluexp-setup-admin/reference-iam-predefined-roles.html["Saiba mais sobre as funções de acesso do NetApp Console ."^]

A remoção do sistema NetApp Console não afeta o sistema E-Series.  Você pode redescobri-lo a qualquer momento.

.Passos
. No menu de navegação, selecione *Armazenamento > Gerenciamento*.
. Na página *Sistemas*, selecione o sistema E-Series.
. Selecione o ícone de menu e selecione *Remover do espaço de trabalho*.
. Selecione *Remover* para confirmar.
